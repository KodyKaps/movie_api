Movie API Documentation

Welcome to the Movie API. This API allows you to retrieve information about movies, including details like title, release year, and rating.

Getting Started

To use the Movie API, follow these steps:

  • Ensure you have Node.js installed
  • Install the necessary dependencies by running npm install
  • Start the server by running node app.js

API Endpoints

Get All Movies

Returns a list of all movies in the database.

GET /api/movies

Get Movie by ID

Returns details of a specific movie by its ID.

GET /api/movies/:id

Example Response


{
  "id": 1,
  "title": "Inception",
  "year": 2010,
  "rating": 8.8
}
